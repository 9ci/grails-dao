Package: gorm.tools.testing

[Groovy] Class TestDataJson

    • gorm.tools.testing.TestDataJson

    • static build methods to wrap grails.buildtestdata.TestData and Jsonify's statics for the json-views. These helpers enable the easy generation of test data in Map form to test methods like the repo's create and update. Note: when using this in unit test the JsonViewSpecSetup should be used to make sure the proper view-tools beans are setup. This is alrady taken care of if using one of the main DataRepoTest or DomainRepoTest traits
