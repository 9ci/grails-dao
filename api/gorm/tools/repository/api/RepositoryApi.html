      • D get(Serializable id, Long version)

        gets and verifies that the entity can eb retrieved and version matches.

        throws:
        gorm.tools.repository.errors.DomainNotFoundException if its not found
        throws:
        org.springframework.dao.OptimisticLockingFailureException if the versions mismatch
        Parameters:
        id - required, the id to get
        version - - can be null. if its passed in then it validates its that same as the version in the retrieved entity.
        Returns:
        the retrieved entity. Will always be an entity as this throws an error if not
