        • Datastore datastore

        • Class<D> domainClass

          The gorm domain class. will generally get set in contructor or using the generic as done in GormRepo.getDomainClass using the GenericTypeResolver

        • boolean enableEvents

          default to true. If false only method events are invoked on the implemented Repository.

          The data binder to use. By default gets injected with EntityMapBinder
