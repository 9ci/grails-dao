        • @Deprecated @CompileDynamic static Criteria filterRange(Map params, def delegate, Closure cast, List keys)

          Does date or money range comparison constraints. Should be able to handle any combination of inequalities. Performs up to 2 comparisons on the value in an AND relationship. Can handle any inequality and any sort of 'between-ness' but is safe from injection of odd operators.

          Parameters:
          params - the parameter map.
          delegate - The delegate from within the createCriteria().list() structure.
          cast - A Closure which can convert the JSON value to the type in the domain object.
          keys - the list of domain attribute names params[key] value needs to be a map which has the following values: minop: Optional. Values: 'eq', 'ne', 'gt', 'ge' minval: Required if minop exists, otherwise ignored. A value which can be converted to the domain type using 'cast' closure. maxop: Optional. Values: 'lt', 'le' maxval: Required if maxop exists, otherwise ignored. A value which can be converted to the domain type using 'cast' closure. The params[key] value needs to have either the min* pair or the max* pair, or both, or the method will have no effect on the query.
